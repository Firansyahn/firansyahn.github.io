About Me

Hallo, Nama saya Naufal dan saya suka mempelajari hal baru tentang dunia IT terutama pada website. Ketertarikan saya dengan website dimulai pada tahun 2016 ketika saya membuat website untuk tempat praktik kerja lapang, dari situ saya belajar banyak tentang HTML & CSS dan juga framework PHP (CI).

Fokus utama saya adalah untuk membuat produk simple mudah diakses serta sesuai dengan trend IT masa kini, dan saya selalu antusias untuk selalu mempelajari dan menerapkan teknologi trend masa kini.
